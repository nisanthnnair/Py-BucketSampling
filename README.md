# Py-BucketSampling
Python based program for computing free energy surface from bucket sampling simulations, based on the original work of Javed et al. J. Chem. Theory Comput. 2024, 20, 19, 8450–8460 (https://doi.org/10.1021/acs.jctc.4c00776). For detailed tutorial see this link: https://sites.google.com/view/the-nnn-group/tutorials/bucket-sampling.
